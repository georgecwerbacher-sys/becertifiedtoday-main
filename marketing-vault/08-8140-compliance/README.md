# DoD 8140 compliance — reference & marketing

Workspace for **official DoD Cyberspace Workforce (8140)** source material and how it maps to Be Certified Today **exam prep** (not clearance, HR tracking, or contract compliance services).

## Drop PDFs here

Place official documents for review in:

**`source-documents/`**

Suggested naming when you add files:

```
source-documents/DoD-8140-{short-title}-{YYYY-MM}.pdf
```

After upload, note the filename in [[marketing-incorporation-plan|marketing incorporation plan]] so Cursor/Obsidian reviews can reference the right doc.

## What this folder is for

| Use | Not for |
|-----|---------|
| Map **work roles → baseline certs** to current/future courses | Claiming we satisfy a specific contract or billet |
| Inform **federal/contractor ad copy** and landing `#exam-audience` sections | Storing classified or CUI material |
| Cross-check messaging against **official 8140 guidance** | Replacing legal/HR sign-off on cert requirements |

4. Log site deploys in [[../06-website-optimization/content-change-log|content change log]].
