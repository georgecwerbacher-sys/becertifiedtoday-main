## Scope

**Dump tracker:** *dumps*, *braindumps*, *actual exam questions*, PDF + VCE/ETE bundles.

**Practice tracker:** ExamCompass-style tests, OpenExamPrep, PrepTIA, CertiMaan, CertBlaster samples, Mastery public pages, Tech Exam Lexicon, etc.

**Out of scope (elsewhere):** CertMaster, Professor Messer, Dion, Udemy **courses**, Pearson/Sybex **books**.
