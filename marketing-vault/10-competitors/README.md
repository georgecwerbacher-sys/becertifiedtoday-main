## Scope rules

**In scope:** domains using *dumps*, *braindumps*, *actual exam questions*, *PDF + test engine*, *VCE/ETE* bundles.

**Out of scope (document elsewhere):** CompTIA CertMaster, Professor Messer, Dion, Udemy courses, Pearson/Sybex books, Boson netsim labs — those are **training**, not dump PDF retailers.
