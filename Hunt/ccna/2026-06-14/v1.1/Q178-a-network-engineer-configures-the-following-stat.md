A network engineer configures the following static route:

  ip route 10.10.0.0 255.255.0.0 GigabitEthernet0/1

What type of static route is this, and what potential issue can it cause on a multi-access network?

- A. Next-hop route; it can cause asymmetric routing
- B. Directly attached static route; it can cause excessive ARP requests because the router must ARP for every destination in the network
- C. Recursive static route; it requires multiple routing table lookups
- D. Fully specified static route; it has no potential issues

**Stated answer (external):** B
