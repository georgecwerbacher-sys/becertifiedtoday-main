A router has the following entries in its routing table for the same destination network:

  D 10.10.0.0/16 [90/2170112] via 172.16.1.1
  O 10.10.0.0/16 [110/65] via 172.16.2.1
  R 10.10.0.0/16 [120/3] via 172.16.3.1

Which route will be installed in the routing table?

- A. The OSPF route via 172.16.2.1 because it has the lowest metric
- B. The RIP route via 172.16.3.1 because it has the fewest hops
- C. The EIGRP route via 172.16.1.1 because it has the lowest administrative distance
- D. All three routes will be installed for load balancing

**Stated answer (external):** C
