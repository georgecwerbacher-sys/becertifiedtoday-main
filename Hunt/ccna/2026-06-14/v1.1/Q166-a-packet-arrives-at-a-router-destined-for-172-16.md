A packet arrives at a router destined for 172.16.50.10. The routing table has no matching entry for this destination, and no default route is configured. What will the router do?

- A. Forward the packet out all interfaces except the one it arrived on
- B. Send the packet to the next-hop router with the lowest IP address
- C. Drop the packet and send an ICMP Destination Unreachable message to the source
- D. Buffer the packet and send an ARP request for the destination

**Stated answer (external):** C
