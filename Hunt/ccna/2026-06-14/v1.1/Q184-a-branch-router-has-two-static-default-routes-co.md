  ip route 0.0.0.0 0.0.0.0 10.1.1.1
  ip route 0.0.0.0 0.0.0.0 10.2.2.1

What will the router do with traffic that matches no specific route?

- A. Use only the first route because it was configured first
- B. Use only the second route because it was configured last

- D. Generate an error and neither route will be installed

**Stated answer (external):** C
