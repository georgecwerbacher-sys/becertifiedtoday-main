Match your prep to how Cisco weights CCNA 200-301

The live CCNA 200-301 exam (v1.1) certifies associate-level networking skills across routing, switching, IP services, security, and automation. Cisco publishes six domains; official guidance puts the heaviest weight on IP connectivity (25%), with network fundamentals and network access (20% each) close behind.

Domain weights below come from Cisco exam guidance. Use them to budget study hours, not as a guarantee of how many questions you will see on a specific attempt. The exam allows up to 120 minutes with multiple-choice, drag-and-drop, and simulation-style items.

Domain Weight What to study
Network fundamentals 20% OSI and TCP/IP models, IPv4/IPv6, cabling, devices, topologies, subnetting basics
Network access 20% VLANs, trunking, basic STP, EtherChannel, wireless concepts, switch port configuration
IP connectivity 25% Routing tables, static routing, OSPF fundamentals, path selection, troubleshooting routing
IP services 10% NAT, DHCP, DNS, NTP, SNMP, syslog, QoS basics
Security fundamentals 15% Device hardening, ACLs, authentication, AAA basics, secure management access
Automation and programmability 10% APIs, controllers, automation concepts, JSON, Python basics, intent-based networking

What to focus on first: network fundamentals and IP connectivity underpin subnetting, routing, and a large share of drag-and-drop or simulation-style prompts. Add network access and security fundamentals next. They show up on both theory and configuration-style questions.

Suggested study order

  1. Network fundamentals (models, addressing, subnetting)
  2. Network access (VLANs, switching, wireless basics)
  3. IP connectivity (routing, OSPF, troubleshooting)
  4. IP services (NAT, DHCP, DNS, NTP)
  5. Security fundamentals (ACLs, device hardening, AAA)
  6. Automation and programmability (APIs, JSON, Python basics)

Under time pressure, leaning hardest into IP connectivity and network fundamentals before you polish lighter domains mirrors where Cisco weights the blueprint. The timed simulation on this site rehearses that mixed multiple-choice, drag-and-drop, and CLI lab pressure across all six domains in one exam-style session.

Source: Cisco CCNA 200-301 exam blueprint, v1.1.
