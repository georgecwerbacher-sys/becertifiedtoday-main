Study path

Security+ SY0-701 blueprint and domain weights

Security+ study lines up with five main knowledge domains. CompTIA publishes the current SY0-701 exam areas (objectives version 5.0) as general security concepts, threats and vulnerabilities, security architecture, security operations, and security program management. Official guidance puts the strongest emphasis on security operations (28%), followed by threats, vulnerabilities, and mitigations (22%) and security program management and oversight (20%).

Security+ study sections

Domain weights below come from the CompTIA Security+ SY0-701 exam objectives document—use them to budget study hours, not as a guarantee of item counts on your specific attempt. The live exam allows up to 90 questions in 90 minutes (multiple-choice and performance-based).

Where to aim your reading and practice

CompTIA Security+ SY0-701 domain weights and study focus areas
Section Weight What to study
1.0 General Security Concepts 12% Security controls, CIA and AAA, zero trust, physical security, cryptography and PKI basics, change management
2.0 Threats, Vulnerabilities, and Mitigations 22% Threat actors, attack vectors, malware and network attacks, vulnerability types, mitigation and hardening
3.0 Security Architecture 18% Cloud and hybrid architecture, identity and access, secure network design, encryption, secure software and supply chain
4.0 Security Operations 28% Asset and vulnerability management, monitoring and alerting, incident response, automation, SIEM, and log analysis
5.0 Security Program Management and Oversight 20% Governance, risk and compliance, policies and awareness, third-party risk, disaster recovery, privacy, and audit

What to focus on first

Start with general security concepts and threats, vulnerabilities, and mitigations—they build the language for everything else. Then layer in security architecture, security operations (the largest domain), and program management for governance, risk, and compliance scenarios.

Study order suggestion

  1. General security concepts (controls, CIA, AAA, cryptography basics)
  2. Threats, vulnerabilities, and mitigations
  3. Security architecture (identity, cloud, network design)
  4. Security operations (monitoring, IR, automation)
  5. Security program management and oversight

Under time pressure, leaning hardest into Security Operations and Threats, Vulnerabilities, and Mitigations before you polish lighter domains mirrors where CompTIA weights the blueprint.
