Advanced Firewall Rule Configurator

You are a network security engineer configuring a perimeter firewall for Acme Corp. The web server is at 192.168.1.10 (internal). The management workstation subnet is 10.0.0.0/24. Apply the security policy by adding the correct ACL rules below.

Policy requirements — checked after grading

  • Allow inbound HTTPS (TCP 443) from any source to the web server
  • Allow inbound SSH (TCP 22) to the web server from the management subnet only
  • Deny all inbound Telnet (TCP 23) from any source
  • Allow outbound DNS (UDP 53) from the internal network 192.168.1.0/24
  • Implicit deny-all (last rule)
