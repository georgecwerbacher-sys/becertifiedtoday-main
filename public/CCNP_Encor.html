Cisco core exam topics by track

Cisco core exam topics depend on the certification track. The most common request is the CCNP Enterprise core exam, while the CCNA core exam has a different scope.

CCNP Enterprise core (ENCOR 350-401) major areas:

  • Architecture
  • Virtualization
  • Infrastructure
  • Network assurance
  • Security
  • Automation

CCNA core exam focus: network fundamentals, network access, IP connectivity, IP services, security fundamentals, and automation/programming.
