NetFlow, SPAN & IP SLA Sim 2

R1 has Flexible NetFlow definitions (use monitor Monitor-R1Flow on Ethernet0/0). Sw1: local SPAN session 12 mirrors VLAN 12 to E1/3. Task 3: choose ICMP echo to PC1 every 300s or HTTP GET version (600s) before you enter the first command — selection locks after R1 Task 1 starts.

Tasks
  1. On R1, apply the pre-built flow monitor on e0/0 for input and output.
  2. On Sw1, SPAN session 12: source VLAN 12 both, destination e1/3; verify with show monitor session 12.
  3. On R1, new IP SLA operation 1: ICMP to 10.12.1.2 every 300s or HTTP GET to 10.10.1.100 every 600s — then schedule life forever start-time now.
  4. Save on R1 and Sw1.

Command sequence (spoiler)

R1 — NetFlow

interface e0/0
 ip flow monitor Monitor-R1Flow input
 ip flow monitor Monitor-R1Flow output

Sw1 — SPAN

monitor session 12 source vlan 12 both
monitor session 12 destination interface e1/3

R1 — IP SLA (ICMP)

ip sla 1
 icmp-echo 10.12.1.2 source-interface Ethernet0/0
 frequency 300
 exit
ip sla schedule 1 life forever start-time now

R1 — IP SLA (HTTP v2)

ip sla 1
 http-get 10.10.1.100
 frequency 600
 exit
ip sla schedule 1 life forever start-time now

Save: copy running-config startup-config on R1, then Sw1.
