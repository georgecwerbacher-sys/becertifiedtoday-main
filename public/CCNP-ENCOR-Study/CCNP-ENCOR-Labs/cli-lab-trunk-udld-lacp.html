Lab complete. E0/0 is a dot1q trunk (VLAN 300) with UDLD aggressive; Port-channel 10 LACP active with trunks allowing 1,300; optional portfast and long pathcost; config saved.

Helper File

Task 1 - Restore SW10 to SW30 trunk on E0/0 (VLAN 300 only)

SW10#conf t
SW10(config)#interface e0/0
SW10(config-if)#switchport trunk encapsulation dot1q
SW10(config-if)#switchport mode trunk
SW10(config-if)#switchport trunk allowed vlan 300
SW10(config-if)#end

Task 2 - Enable aggressive UDLD on E0/0

SW10#conf t
SW10(config)#interface e0/0
SW10(config-if)#udld port aggressive
SW10(config-if)#end

Task 3 - Rebuild LACP on E0/2-3 for Port-channel10

SW10#conf t
SW10(config)#no interface po10
SW10(config)#interface range e0/2 - 3
SW10(config-if-range)#switchport trunk encapsulation dot1q
SW10(config-if-range)#switchport mode trunk
SW10(config-if-range)#switchport trunk allowed vlan 1,300
SW10(config-if-range)#channel-group 10 mode active
SW10(config-if-range)#end

Optional hardening

SW10#conf t
SW10(config)#interface e0/1
SW10(config-if)#switchport mode access
SW10(config-if)#spanning-tree portfast
SW10(config-if)#exit
SW10(config)#spanning-tree pathcost method long
SW10(config)#end

Save

SW10#copy running-config startup-config
