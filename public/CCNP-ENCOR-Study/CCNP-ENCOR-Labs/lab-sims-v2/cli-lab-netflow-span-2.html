Tasks

The Operations team started configuring several monitoring activities. Complete the configurations for the tasks below.

  1. Enable Flexible NetFlow on R1 E0/0 in both directions using the pre-configured flow monitor.
  2. Configure the switch port analyzer on Sw1 and mirror all VLAN 12 traffic to interface E1/3 using session number 12.
  3. Configure a basic IP SLA ICMP echo operation on R1 to ping PC1 every 300 seconds.

    Or (Version 2): Configure an HTTP GET operation on R1 to 10.10.1.100 every 600 seconds.

Lab complete. NetFlow monitor on e0/0, SPAN 12 verified, IP SLA 1 scheduled, configs saved.

Helper File

Task 1. Enable Flexible NetFlow on R1 E0/0 in both directions using the pre-configured flow monitor.

On R1:

R1(config)#interface e0/0
R1(config-if)# ip flow monitor Monitor-R1Flow input
R1(config-if)# ip flow monitor Monitor-R1Flow output

Task 2. Configure the switch port analyzer on Sw1 and mirror all VLAN 12 traffic to interface E1/3 using session number 12.

On Sw1:

Sw1(config)#monitor session 12 source vlan 12 both
Sw1(config)#monitor session 12 destination interface e1/3

Task 3. Configure a basic IP SLA ICMP echo operation on R1 to ping PC1 every 300 seconds.

The IP address of PC1 is 10.12.1.2

On R1:

R1(config)# ip sla 1
R1(config-ip-sla)# icmp-echo 10.12.1.2 source-interface Ethernet0/0
R1(config-ip-sla-echo)#frequency 300
R1(config-ip-sla-echo)#exit
R1(config)# ip sla schedule 1 life forever start-time now
