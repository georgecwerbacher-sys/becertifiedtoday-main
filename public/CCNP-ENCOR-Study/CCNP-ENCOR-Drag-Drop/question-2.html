On a phone or tablet, tap a chip to pick it up, then tap a drop zone (or another chip to swap).

Drag and drop the code snippets from the bottom onto the blanks in the JSON template to build the extended ACL request. All four snippets are used.

Fill all four blanks, then click Check Answer.

JSON request template

"ip": {
  "access-list": {
    "ios-acl:extended": {
      "ios-acl:name": "ato",
      "ios-acl:": {
        "ios-acl:sequence": "111111",
        "ios-acl:ace-rule": {
          "ios-acl:action": "",
          "ios-acl:protocol": "",
          "ios-acl:": ""
        }
      }
    }
  }
}

Drag each token into the matching blank.
