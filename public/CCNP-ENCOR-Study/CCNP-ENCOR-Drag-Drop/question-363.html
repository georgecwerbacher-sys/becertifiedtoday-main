On a phone or tablet, tap a chip to pick it up, then tap a drop zone (or another chip to swap).

Drag and drop the code snippets from the bottom onto the blanks in the JSON request template to construct a RESTCONF-style body that configures policy-based routing (PBR). All four snippets are used.

Fill all four blanks, then click Check Answer.

JSON request template

{
  "ios-route-map:route-map": [
    {
      "ios-route-map:name": "PBR",
      "ios-route-map:route-map-without-order-seq": [
        {
          "ios-route-map:operation": "",
          "ios-route-map:set": {
            "ios-route-map:": {
              "ios-route-map:": {
                "ios-route-map:": "10.10.10.10"
              }
            }
          }
        }
      ]
    }
  ]
}
