Which Python snippet stores the data structure of the device in JSON format?

Refer to the exhibit.

Python
import json
Devices={'Switches':
    [
      {'name': 'AccSw1', 'ip': '2001:db8:1:fffe::1'},
      {'name': 'AccSw2', 'ip': '2001:db8:1:fffe::2'}
    ],
    'Routers':
      {'CE1': {'ip': '2001:db8:1:ffff::1'},
       'CE2': {'ip': '2001:db8:1:ffff::2'}
      }
}

Options A through D (same as answer choices).

Option A
with open("devices.json", "w") as OutFile:
    Devices = json.load(OutFile)
Option B
OutFile = open("devices.json", "w")
json.dump(Devices, OutFile)
OutFile.close()
Option C
with open("devices.json", "w") as OutFile:
    json.dumps(Devices)
Option D
OutFile = open("devices.json", "w")
OutFile.write(str(Devices))
OutFile.close()
