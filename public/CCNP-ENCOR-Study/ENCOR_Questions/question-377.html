A BeCertifiedToday.com engineer must configure PAT to provide internet access to all users by using one global address for many local addresses. Which command set completes the configuration?

RouterA
RouterA(config)#ip nat pool cisco 193.64.64.1 193.64.64.15 netmask 255.255.255.240
RouterA(config)#interface g0/0/0
RouterA(config-if)#ip nat inside
RouterA(config-if)#interface serial0/0/0
RouterA(config-if)#ip nat outside
RouterA(config-if)#exit
