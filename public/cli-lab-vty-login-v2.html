Device in prompt:

Exams may ask you to configure R1 or R2. Choose Device in prompt under the topology, then Open Router CLI; prompts match the device you pick. Commands are the same. You start in global configuration on that device.

Three results:
  1. Same as Version 1: local user SuperAdmin / SuperUser, privilege 15 (secret or password).
  2. VTY 0–4: login local, transport input telnet rlogin (not SSH in this variant).
  3. AUX line: exec-timeout 20 0 on the auxiliary port (20 minutes inactivity).
Finish: leave configuration (end), then copy running-config startup-config (or copy run start). After the VTY block, use exit to return to global config before line aux 0.
