On a phone or tablet, tap a chip to pick it up, then tap a drop zone (or another chip to swap).

Drag and drop the Cisco IOS DNS commands from the left onto their effects on the right.

Commands

ip domain-name
ip domain-lookup
ip name-server
show hosts
ip host switch_1 192.168.0.1

Effects

enables host-to-IP-address translation

adds an entry to the host table

displays address-mapping information

specifies the IP address of the DNS server

completes the FQDN of the DNS server
