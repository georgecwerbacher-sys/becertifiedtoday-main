Command sequence (spoiler)

  3. tunnel source Ethernet0/0 (or e0/0) → tunnel destination 209.165.200.230 → no shutdown
  4. interface Ethernet0/1 → vrf forwarding Finance → ip address 10.22.22.1 255.255.255.252 → no shutdown
  5. ip route vrf Finance 10.10.111.0 255.255.255.0 10.10.10.1
  6. ip route vrf Finance 10.10.222.0 255.255.255.0 10.22.22.2
  7. end → copy running-config startup-config
