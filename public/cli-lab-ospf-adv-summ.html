Command sequence (spoiler)

R1

  1. router ospf 1 → exit
  2. interface loopback0 → ip ospf 1 area 0
  3. interface ethernet0/0 (or int e0/0) → ip ospf 1 area 0
  4. interface ethernet0/1 → ip ospf 1 area 0 → end
  5. show ip ospf interface brief → show ip route ospf

R2: router ospf 1 → area 10 range 10.1.0.0 255.255.0.0 → end

R3: router ospf 1 → area 20 range 10.2.0.0 255.255.0.0 → end
